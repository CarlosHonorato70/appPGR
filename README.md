# 🔷 Black Belt - Plataforma Integrada

Plataforma para gestão de riscos psicossociais, precificação inteligente e geração de propostas.

## 🚀 Quick Start

### Com Docker (Recomendado)
\\\ash
docker-compose up -d
\\\

Acesse:
- **Frontend**: http://localhost:8501
- **Backend**: http://localhost:3000
- **MySQL**: localhost:3306 (user: root, password: root)

### Localmente

**Terminal 1 - Backend:**
\\\ash
cd backend
npm install
npm run dev
\\\

**Terminal 2 - Frontend:**
\\\ash
cd streamlit
pip install -r requirements.txt
streamlit run app.py
\\\

## 📋 Funcionalidades

✅ **Precificação Inteligente**
- Cálculo de hora técnica
- Suporte a 4 regimes tributários

✅ **Gestão de Propostas**
- Criação e gerenciamento
- Exportação em PDF

✅ **Avaliação de Riscos NR-01**
- Conformidade com norma
- Fatores psicossociais

✅ **Dashboard Analítico**
- Visualização de métricas
- Relatórios em tempo real
